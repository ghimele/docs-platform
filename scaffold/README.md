# Documentation

This folder contains all documentation for the project.
It is the single source of truth for architecture, design decisions, specifications, and conventions.

Agents: read `AGENTS.md` at the repository root before performing any task.

## Document Status Lifecycle

```text
Draft → Active → Deprecated → Superseded
```

- **Draft** — work in progress, not yet authoritative
- **Active** — current and authoritative
- **Deprecated** — still valid but no longer recommended
- **Superseded** — replaced by another document (link provided)
