# Templates

This folder contains document templates.

Do **not** treat files in this folder as real documents.
They are starting points for new documentation only.

---

## Available Templates

| Template | Use for |
| -------- | ------- |
| [adr.md](adr.md) | Architectural Decision Records |
| [rfc.md](rfc.md) | Requests for Comments (open proposals) |
| [architecture.md](architecture.md) | Architecture documents |
| [design.md](design.md) | Design documents |
| [spec.md](spec.md) | Functional and technical specifications |

---

## Usage

Copy the relevant template to the appropriate folder, rename it following
the naming conventions in `../STYLE.md`, then fill in all sections.
Remove any placeholder comments (lines starting with `<!--`).
