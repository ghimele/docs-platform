# Requests for Comments (RFCs)

**Status:** Active  
**Last updated:** 2026-03-30  

This folder contains open proposals under active discussion.

RFCs are **mutable** — they can be freely edited during the discussion phase.
Once a decision is made, an RFC graduates to an ADR in `../adr/` and is closed here
by marking its status as `Accepted` or `Rejected`.

---

## How to Create an RFC

1. Assign the next sequential ID by scanning this folder
2. Copy `../templates/rfc.md` and rename it `RFC-NN-short-title.md`
3. Fill in all sections
4. Add an entry to the index table below
5. Share for discussion

## Graduation to ADR

When an RFC reaches a decision:

1. Create a new ADR in `../adr/` using `../templates/adr.md`
2. Reference the RFC number in the ADR's context section
3. Update the RFC status to `Accepted` or `Rejected` and add a link to the ADR

## Status Values

| Status | Meaning |
| ------ | ------- |
